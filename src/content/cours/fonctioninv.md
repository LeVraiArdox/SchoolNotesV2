# Fonction inverse


Pour tout nombre réel $x$, on note $x^{-1}$ ou $\frac{1}{x}$ le nombre réel tel que $x \times x^{-1} = 1$.

On peut remarquer que la fonction inverse n'est pas définie en $0$.

La fonction inverse est définie sur $\mathbb{R}^*$.
